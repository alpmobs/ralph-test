# Shopify Test

**Objective**: Build a simple custom Shopify homepage based on the provided layout reference. The page must be section-based, responsive, and easy for a non-developer admin to update from the Shopify Theme Editor. The most important requirement is that every visible content area is configurable, including text, images, buttons, links, product selections, collection selections, reviews, footer content, section spacing, and basic display options.

## List of the Sections to Build
> #### sections/custom-homepage-hero.liquid: 
> The hero banner I created was inspired by premium themes, with a more dramatic design to better capture the user’s attention. I also added multiple schema settings and customization options to allow flexible content positioning based on the selected background image.

> #### sections/custom-homepage-benefits.liquid: 
> The Benefits section supports custom image icons and provides flexible settings, making it easy to update and customize the content as needed.  

> #### sections/custom-homepage-featured-products.liquid
> The Featured Products section includes additional options and customization settings to provide greater flexibility and create a more polished, visually appealing layout. 

> #### sections/custom-homepage-collections.liquid
> The Collections section is different from the other sections because it supports manual configuration and allows full customization of the collection cover images and details.

> #### sections/custom-homepage-reviews.liquid
> The Reviews section uses a unique approach that allows reviews to be linked to specific products while also providing the ability to customize each reviewer’s image.


> #### sections/custom-homepage-footer.liquid:
> The Footer section includes all the necessary fields and is designed to make updating and maintaining the content easier.
